# 要件定義 - 音響合成ウェーブフォームエディター

## アプリ番号
[00000077]

## アプリ名
音響合成ウェーブフォームエディター

## 概要
Web Audio APIを活用したリアルタイム音響合成システム。
複数のオシレーター、ADSRエンベロープ、フィルターを組み合わせて
音色を自由に創作し、波形を視覚的に確認できるWebアプリケーション。
音響合成の理論と実践を学習できる教育的ツール。

## 機能要件

### 1. オシレーターシステム
- **波形タイプ**
  - サイン波（純音、基本波形）
  - 方形波（矩形波、豊富な倍音）
  - ノコギリ波（全倍音、明るい音色）
  - 三角波（奇数倍音、柔らかい音色）
- **周波数制御**
  - 基本周波数: 20Hz〜20kHz
  - オクターブシフト: ±3オクターブ
  - ファインチューニング: ±100セント
- **マルチオシレーター**
  - 最大4種類の波形同時合成
  - 各オシレーターの音量バランス調整
  - ミックス比率のリアルタイム制御

### 2. ADSRエンベロープ制御
- **Attack（アタック）**: 0〜5秒
  - 音の立ち上がり時間
  - linearRampToValueAtTimeで滑らかな変化
- **Decay（ディケイ）**: 0〜5秒  
  - ピークからサスティンレベルまでの減衰時間
  - exponentialRampToValueAtTimeで自然な減衰
- **Sustain（サスティン）**: 0〜100%
  - キー押下中の維持レベル
  - 長時間音を保持する際の音量
- **Release（リリース）**: 0〜10秒
  - キー離し後の減衰時間
  - 音の自然な消失をシミュレート

### 3. フィルターシステム
- **フィルタータイプ**
  - ローパスフィルター（LPF）: 高域カット
  - ハイパスフィルター（HPF）: 低域カット  
  - バンドパスフィルター（BPF）: 特定帯域通過
- **フィルターパラメータ**
  - カットオフ周波数: 20Hz〜20kHz
  - レゾナンス（Q値）: 0.1〜30
  - フィルター強度: 0〜100%
- **動的制御**
  - リアルタイムパラメータ変更
  - フィルタースイープ効果
  - 複数フィルターのチェーン接続

### 4. 波形可視化システム
- **リアルタイム描画**
  - Canvas APIによる高速描画
  - 60fps での滑らかな更新
  - AnalyserNodeによる正確な波形解析
- **表示機能**
  - 時間軸波形表示（オシロスコープ風）
  - 振幅スケール自動調整
  - グラデーション効果による美しい描画
- **視覚的フィードバック**
  - 周波数成分の色分け表示
  - エンベロープ変化の可視化
  - フィルター効果の波形変化表示

### 5. 演奏インターフェース
- **バーチャルキーボード**
  - 2オクターブ（C3〜B4）のキー配置
  - マウスクリック演奏対応
  - キーボードショートカット（QWERTY配列）
- **音階制御**
  - 平均律12音階システム
  - 正確な音程計算（A4=440Hz基準）
  - 複数音同時発音対応（ポリフォニー）
- **演奏支援**
  - キープレス視覚フィードバック
  - 音程表示（ノート名＋オクターブ）
  - サスティンペダル機能

### 6. プリセットシステム
- **標準プリセット**
  - Bass（ベース音色）: サイン波＋LPF
  - Lead（リード音色）: ノコギリ波＋HPF
  - Pad（パッド音色）: 三角波＋長いリリース
  - Pluck（プラック音色）: 方形波＋短いアタック
  - Sweep（スイープ音色）: フィルタースイープ効果
- **カスタムプリセット**
  - 現在設定の保存機能
  - プリセット名の編集
  - お気に入り登録システム

### 7. パラメータ制御
- **スライダーUI**
  - 全パラメータのリアルタイム調整
  - 視覚的な値表示（数値＋バー）
  - マウス＋キーボード操作対応
- **自動化システム**
  - LFO（低周波発振器）によるパラメータ変調
  - ランダム値生成機能
  - パラメータアニメーション

### 8. 教育機能
- **理論説明**
  - 各波形の特性解説
  - ADSRエンベロープの音楽的効果
  - フィルターの周波数特性
- **実験機能**
  - パラメータ変更の即座の音響フィードバック
  - 波形比較機能
  - 音響理論の実践的学習

## 非機能要件

### 1. パフォーマンス要件
- **応答性**: パラメータ変更0.1秒以内の音響反映
- **描画性能**: 60fps での波形可視化
- **音質**: Web Audio API標準（44.1kHz/16bit相当）
- **レイテンシ**: 音響生成遅延20ms以下

### 2. ブラウザ互換性
- **対応ブラウザ**: Chrome, Firefox, Safari, Edge最新版
- **Web Audio API**: フル機能サポート必須
- **Canvas API**: 2D描画機能必須
- **JavaScript**: ES6+機能使用

### 3. UI/UX要件
- **レスポンシブデザイン**: デスクトップ/タブレット対応
- **直感的操作**: 音響知識なしでも基本操作可能
- **視覚的フィードバック**: 全操作に即座の視覚反応
- **アクセシビリティ**: キーボード操作完全対応

### 4. 技術仕様
- **フロントエンド**: 純粋JavaScript（ES6+）
- **音響処理**: Web Audio API
- **描画**: Canvas API
- **UI**: CSS3（Grid Layout, Flexbox）
- **外部依存**: なし（スタンドアローン）

## 制約事項
- **ブラウザ制限**: Autoplay Policy準拠（ユーザーアクション必須）
- **メモリ使用量**: 音響バッファサイズ最適化
- **CPU負荷**: リアルタイム処理の効率化
- **音響品質**: Web Audio APIの仕様範囲内

## 使用シナリオ
1. **音響学習**: 波形とパラメータの関係学習
2. **音色制作**: 楽曲制作用の音色創作
3. **実験**: 音響合成理論の実践的確認
4. **デモンストレーション**: 音響技術の視覚的プレゼンテーション

## 成功基準
- 全オシレーター・フィルター・エンベロープが正確に動作
- リアルタイム波形描画が滑らかに表示
- プリセット切り替えが瞬時に反映
- 音響パラメータ変更の即座の音質変化
- 教育的価値の高い直感的UI実現
