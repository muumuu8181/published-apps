# 乱数生成アルゴリズムベンチマークツール 要件定義書

## アプリID
app-00000080-qypir8

## 概要
複数の乱数生成アルゴリズムの性能と品質を比較・評価するWebベースのベンチマークツール

## 機能要件

### 1. 乱数生成アルゴリズム
- Mersenne Twister (MT19937)
- Xorshift (32/64/128ビット版)
- PCG (Permuted Congruential Generator)
- LCG (Linear Congruential Generator)
- JavaScript標準 Math.random()

### 2. アルゴリズム選択
- ドロップダウンメニューからアルゴリズムを選択
- 複数アルゴリズムの同時選択（バッチ比較モード）

### 3. パラメータ設定
- シード値のカスタム入力
- 生成シーケンス長の指定（1～1,000,000個）
- サンプリング間隔の設定

### 4. 統計テスト
- 均一性テスト（カイ二乗検定）
- 独立性チェック（自己相関）
- モノビットテスト
- ランテスト
- p値計算と表示

### 5. 可視化機能
- ヒストグラム（分布表示）
- 散布図（2D/3D）
- フラクタル風パターンアニメーション
- リアルタイムグラフ更新

### 6. パフォーマンス測定
- 生成速度（numbers/sec）
- メモリ使用量
- CPU使用率推定

### 7. データエクスポート
- CSV形式での結果出力
- 統計レポートのJSON出力
- グラフの画像保存

### 8. カスタムテスト
- ユーザー定義の統計関数
- JavaScriptコードの動的実行
- カスタムテスト結果の統合表示

### 9. UI/UX
- レスポンシブデザイン
- ダーク/ライトテーマ切り替え
- リアルタイムプログレス表示
- ツールチップによる説明

## 非機能要件

### パフォーマンス
- 生成速度: 1M numbers/sec以上
- UI応答時間: 100ms以内
- フレームレート: 60fps維持

### 統計精度
- p値計算精度: 0.95以上
- 誤差率: 0.01未満

### 互換性
- Chrome/Firefox/Safari最新版対応
- モバイル端末対応
- タッチ操作対応

## 技術仕様
- フロントエンド: HTML5, CSS3, JavaScript (ES6+)
- グラフライブラリ: Canvas API
- 統計計算: 純粋JavaScript実装
- アニメーション: requestAnimationFrame
- ワーカー: Web Workers（重い計算用）
