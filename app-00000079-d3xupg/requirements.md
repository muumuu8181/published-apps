# 要件定義 - ベクトル場流線シミュレーター

## アプリ番号
[00000079]

## アプリ名
ベクトル場流線シミュレーター

## 概要
2D/3Dベクトル場を数学関数で定義し、流線トレースアルゴリズム（Runge-Kutta法）で粒子軌道をリアルタイム描画するWebアプリケーション。
各種物理場のシミュレーションを美しく視覚化し、インタラクティブに操作できる。

## 機能要件

### 1. ベクトル場定義
- **実装済みベクトル場**
  - 渦場 (Vortex): dx/dt = -y, dy/dt = x
  - 鞍点場 (Saddle): dx/dt = x, dy/dt = -y
  - 源点場 (Source): dx/dt = x, dy/dt = y
  - 吸込み場 (Sink): dx/dt = -x, dy/dt = -y
  - 一様場 (Uniform): dx/dt = c, dy/dt = 0
  - 双極子場 (Dipole): 3xy/r³, (2y²-x²)/r³
  - 波動場 (Waves): sin/cos関数組み合わせ
  - 螺旋場 (Spiral): r・cos(θ+π/2), r・sin(θ+π/2)
  - 磁場 (Magnetic): -y/r², x/r²
  - 乱流場 (Turbulence): 時間依存複合関数

### 2. 数値積分アルゴリズム
- **Runge-Kutta 4次法**による高精度軌道計算
- **軌道精度**: 誤差 < 0.01
- **適応的ステップサイズ**: 0.001〜0.05の範囲で調整可能

### 3. 粒子システム
- **初期条件設定**: 最大50粒子の同時シミュレーション
- **位置・速度カスタム**: クリック/ドラッグでの粒子追加
- **軌跡長調整**: 10〜500ステップの範囲で設定可能

### 4. 場のパラメータ調整
- **強度スライダー**: 0.1〜5.0の範囲で場の強度調整
- **場の中心移動**: 右クリックで場の中心を移動
- **リアルタイム更新**: パラメータ変更の即座反映

### 5. 視覚化機能
- **グラデーショントレイル**: 軌跡に美しいグラデーション効果
- **輝度アニメーション**: 粒子本体の光る効果
- **ベクトル場表示**: 背景に矢印でベクトル場を可視化
- **フェード効果**: 軌跡の自然な消失

### 6. プリセットシナリオ
- **渦巻銀河モード**: 螺旋場 + 高強度 + 30粒子
- **磁気双極子モード**: 磁場 + 中強度 + 25粒子
- **流体攪拌モード**: 乱流場 + 低強度 + 40粒子
- **重力井戸モード**: 吸込み場 + 低強度 + 35粒子
- **電場モード**: 双極子場 + 高強度 + 20粒子
- **海流モード**: 波動場 + 中強度 + 45粒子

### 7. インタラクティブ操作
- **クリック**: 粒子追加
- **ドラッグ**: 複数粒子の連続追加
- **右クリック**: 場の中心移動
- **ズーム/パン**: マウスホイール/ドラッグ操作
- **キーボードショートカット**: スペース（開始/停止）、R（リセット）

### 8. スナップショット機能
- **画像出力**: 現在の場状態をPNG形式で保存
- **高解像度対応**: キャンバスサイズに応じた出力
- **ファイル名自動生成**: タイムスタンプ付きファイル名

### 9. パフォーマンス要件
- **フレームレート**: 60fps維持
- **シミュレーション時間**: 1秒/100ステップ
- **軌道精度**: 誤差 < 0.01
- **解像度**: フルHD対応
- **最大粒子数**: 50粒子で安定動作

### 10. UI/UX要件
- **コントロールパネル**: 半透明で折りたたみ可能
- **リアルタイム表示**: FPS・粒子数の動的表示
- **数式表示**: 選択したベクトル場の数学的表現
- **レスポンシブ対応**: モバイル〜デスクトップ対応

## 非機能要件
- **ブラウザ互換性**: Chrome, Firefox, Safari, Edge最新版
- **パフォーマンス**: GPU最適化想定の効率コード
- **メモリ使用量**: 100MB以下
- **読み込み時間**: 3秒以内
- **数値安定性**: 長時間実行での発散防止

## 技術仕様
- **言語**: 純粋JavaScript（ES6+）
- **描画**: Canvas API 2D Context
- **数値計算**: Runge-Kutta 4次法
- **アニメーション**: requestAnimationFrame
- **メモリ管理**: 軌跡データの効率的管理
