# 要件・仕様書: お金管理システム

## 元要求
- 収入と支出を入力できること
- 入力された値は後から編集できること
- CSVとしてデータをDLできること

## 解釈した仕様

### 基本機能
1. **収入・支出入力機能**
   - 金額、カテゴリ、日付、メモの入力
   - 収入カテゴリ: 給与、副業、投資、その他
   - 支出カテゴリ: 食費、交通費、光熱費、娯楽、その他

2. **データ編集機能**
   - 既存レコードの編集（モーダルウィンドウ使用）
   - レコードの削除機能
   - リアルタイム更新

3. **データ表示機能**
   - 総収入、総支出、残高の表示
   - 取引履歴テーブル（日付降順）
   - 残高に応じた色分け表示

4. **CSV出力機能**
   - 全取引データのCSV形式エクスポート
   - ファイル名に日付を含める
   - UTF-8エンコーディング対応

### UI/UX設計
- レスポンシブデザイン（モバイル対応）
- モダンなグラデーションデザイン
- 直感的なアイコン使用
- ホバーエフェクトとアニメーション

## 技術的制約・判断

### 使用技術
- HTML5 + CSS3 + Vanilla JavaScript
- ローカルストレージでのデータ永続化
- CSVダウンロード機能（Blob API使用）

### データ構造
```javascript
{
  type: 'income' | 'expense',
  amount: number,
  category: string,
  date: string,
  note: string,
  timestamp: string
}
```

### ブラウザ対応
- モダンブラウザ（Chrome, Firefox, Safari, Edge）
- localStorage API対応必須
- Blob API対応必須
