# ベクトル場流線シミュレーター 要件定義書

## アプリID
app-00000079-ps4d7s

## 概要
2D/3Dベクトル場を数学関数で定義し、Runge-Kutta法による流線トレースで粒子軌道をリアルタイム描画するシミュレーター

## 機能要件

### 1. ベクトル場定義
- 2Dベクトル場: F(x,y) = [Fx(x,y), Fy(x,y)]
- 3Dベクトル場: F(x,y,z) = [Fx(x,y,z), Fy(x,y,z), Fz(x,y,z)]
- 数学関数による場の定義（curl/divergence計算対応）
- プリセット場10種（磁場、流体力学モデル等）

### 2. 粒子システム
- 初期条件設定（最大50粒子）
- 位置/速度のカスタム設定
- リアルタイム軌道描画
- 粒子の色・サイズ設定

### 3. 流線トレースアルゴリズム
- Runge-Kutta法（4次精度）
- ステップサイズ自動調整
- 軌道精度保証（誤差<0.01）
- 軌道長制限設定

### 4. 場パラメータ調整
- 強度係数スライダー
- 渦巻き係数調整
- 時間スケール変更
- 減衰係数設定

### 5. 視覚化機能
- リアルタイム描画（100ステップ/秒）
- トレイルにグラデーション効果
- 輝度アニメーション
- フルHD解像度対応

### 6. インタラクティブ機能
- 3Dカメラコントロール
- ズーム/回転ビュー
- パン操作
- 視点リセット

### 7. スナップショット機能
- 現在状態の画像出力
- 高解像度保存
- PNG/SVG形式対応

### 8. 最適化機能
- GPUアクセラレーション想定コード
- Web Workers活用
- メモリ効率化
- フレームレート最適化

### 9. プリセットシナリオ
1. 電磁場モデル
2. 流体渦モデル
3. 引力場
4. 斥力場
5. 振動場
6. ローレンツアトラクター風
7. ヴァンデルポール振動子
8. ダフィング振動子
9. 重力井戸
10. 複合場

## 非機能要件

### パフォーマンス
- シミュレーション時間: 1秒/100ステップ
- 軌道精度: 誤差<0.01
- 描画解像度: フルHD（1920x1080）
- フレームレート: 60fps維持

### 対応環境
- Chrome/Firefox/Safari最新版
- WebGL 2.0対応
- モバイル端末対応（タッチ操作）

### UI/UX
- レスポンシブデザイン
- 直感的なコントロール
- リアルタイムプレビュー
- エラーハンドリング

## 技術仕様
- フロントエンド: HTML5, CSS3, JavaScript (ES6+)
- 3D描画: WebGL / Three.js
- 数値計算: 純粋JavaScript実装
- 並列処理: Web Workers
- UI: Canvas API + HTML UI
